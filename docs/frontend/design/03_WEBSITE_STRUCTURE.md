### 3.2 주요 섹션 구성

#### 📍 Single Page Application (SPA) 구조

**전체 페이지 플로우:**
```
Hero Section (첫인상)
    ↓
협회 소개 (신뢰 구축)
    ↓
숫자로 보는 협회 (데이터)
    ↓
강사가 되면 달라지는 것들 (가치 제안)
    ↓
가입 프로세스 (How it Works)
    ↓
성공 사례 (사회적 증거)
    ↓
FAQ (불안 해소)
    ↓
가입 CTA (행동 유도)
    ↓
Footer (정보/링크)
```

#### 상세 섹션 분석

**1. Hero Section** 
```yaml
목적: 3초 안에 가치 전달
높이: 100vh (전체 화면)
요소:
  - 메인 슬로건
  - 서브 캐치프레이즈
  - Primary CTA (가입 신청)
  - Secondary CTA (더 알아보기)
  - 배경: 등불/빛 이미지 + 그라데이션
```

**2. 협회 소개 (About)**
```yaml
목적: 신뢰 구축, 정체성 확립
레이아웃: 2-Column (텍스트 + 이미지)
요소:
  - 협회 비전/미션
  - 설립 배경
  - 핵심 가치 3가지
  - 로고/심볼 의미
```

**3. 숫자로 보는 협회 (Stats)**
```yaml
목적: 사회적 증거, 규모 강조
레이아웃: 4-Column Grid
요소:
  - 등록 강사 수 (카운터 애니메이션)
  - 파견 건수
  - 수요처 수
  - 강사 평균 만족도
시각: 큰 숫자 + 아이콘 + 짧은 설명
```

**4. 강사가 되면 달라지는 것들 (Benefits)**
```yaml
목적: 핵심 가치 제안
레이아웃: 3-Column Cards
카드 구성:
  
  Card 1: 💼 안정적인 강의 기회
    - 연간 평균 강의 건수
    - 수요처 다양성
    - 지속적 매칭
  
  Card 2: 💰 투명한 수익 보장
    - 강사료 체계
    - 정산 프로세스
    - 수수료 안내
  
  Card 3: 📈 지속적 성장 지원
    - 교육 프로그램
    - 커리큘럼 제공
    - 커뮤니티 네트워킹
  
  Card 4: 🎓 전문가 브랜딩
    - 협회 인증
    - 프로필 관리
    - 포트폴리오 구축
```

**5. 가입 프로세스 (How it Works)**
```yaml
목적: 간편함 강조, 불안 해소
레이아웃: Horizontal Stepper (4단계)
시각: 숫자 + 아이콘 + 화살표

Step 1: 온라인 신청 (3분)
  - 간단한 정보 입력
  - 경력 증빙 첨부

Step 2: 서류 검토 (1-2일)
  - 협회 담당자 검토
  - 이메일 안내

Step 3: 승인 & 가입
  - 승인 완료 알림
  - 관리 솔루션 접근

Step 4: 첫 강의 매칭
  - 맞춤 기회 제안
  - 강의 시작
```

**6. 성공 사례 (Success Stories)**
```yaml
목적: 사회적 증거, 감정적 연결
레이아웃: Carousel (3개씩 표시)
카드 구성:
  - 강사 사진 (원형)
  - 이름 + 전문 분야
  - 핵심 성과 (숫자)
  - 한 줄 후기 (따옴표)
예시:
  "협회 가입 후 6개월간 15회 강의, 월 평균 300만원 수익"
  - 김OO 강사 (전 개발자)
```

**7. FAQ**
```yaml
목적: 불안 요소 제거
레이아웃: Accordion (펼침/접기)
카테고리:
  - 가입 조건
  - 강사료 & 정산
  - 강의 준비
  - 계약 & 운영
각 항목: 질문 + 상세 답변
```

**8. 가입 CTA (Final Push)**
```yaml
목적: 최종 전환
레이아웃: Full-Width Section
요소:
  - 임팩트 있는 문구
  - Large CTA Button
  - 보조 정보 (전화, 이메일)
배경: Primary Color Gradient
```

// 숫자 카운팅 애니메이션
function animateCounter(element, target, duration) {
  let start = 0;
  const increment = target / (duration / 16);
  
  const timer = setInterval(() => {
    start += increment;
    if (start >= target) {
      element.textContent = target;
      clearInterval(timer);
    } else {
      element.textContent = Math.floor(start);
    }
  }, 16);
}

// Intersection Observer (스크롤 시 트리거)
const observer = new IntersectionObserver((entries) => {
